# E-commerce платформа для тех, кто носит тьму с достоинством.

##  Особенности, от которых мурашки по коже

- **Полнофункциональная корзина** — добавление, удаление, изменение количества
- **Регистрация и профиль** — с аватаром, историей заказов и избранным
- **Категории товаров** — «Кольца», «Кулоны», «Браслеты»
- **Django Admin** — кастомизирован под продавца: управление товарами, заказами, скидками
- **Пагинация** — удобная навигация по каталогу 
- **Чистый HTML/CSS/JS** — без тяжелых фреймворков, но с атмосферой
